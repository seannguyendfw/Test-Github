# Mô tả dự án

Đây là một kho chứa các tệp SQL dùng để thử nghiệm và lưu trữ truy vấn.

## Mục tiêu
- Lưu trữ các script SQL để kiểm tra và minh họa.
- Dễ mở và chạy bằng công cụ quản lý cơ sở dữ liệu thông thường.

## Cấu trúc dự án
- `SQL testing 2.sql` — Tập hợp các truy vấn thử nghiệm (phiên bản 2).
- `SQL Testing.sql` — Tập hợp các truy vấn thử nghiệm (phiên bản gốc).

## Hướng dẫn sử dụng
1. Mở tệp `.sql` bằng công cụ SQL mà bạn dùng (SQL Server Management Studio, MySQL Workbench, DBeaver, v.v.).
2. Kiểm tra đầu file để biết chú thích hoặc thông số cấu hình (nếu có).
3. Chạy từng khối truy vấn hoặc toàn bộ file tùy mục đích.

Lưu ý: Kho này không kèm dữ liệu mẫu hay cấu hình kết nối — bạn cần kết nối với cơ sở dữ liệu phù hợp trước khi chạy.

## Liên hệ
- Nếu cần trợ giúp hoặc chỉnh sửa README, phản hồi trực tiếp trong repository này.
